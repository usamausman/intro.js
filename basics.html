Basics

Introduction

JavaScript, or JS, is defined as an interpreted scripting language, but what does this mean?

An interpreted language is parsed and run by an interpreter, so it does not need to be compiled, like C or Java. Also, as a scripting language, it is very easy to use for simple tasks such as running animations and adding functionality to webpages.

However, JavaScript is not just for webpages. Through the use of JavaScript runtimes such as node.js, it can be used to host websites and even runs applications, such as Discord and Visual Studio Code.

This introduction to JS will cover the basic principles and concepts you can use to write your very own programs.

Math

JavaScript supports a number of basic math operations:

2 + 3 // -> 5
7 - 4 // -> 3
5 * 7 // -> 35
2 + 3 * 7 - 4 // -> 19
(2 + 3) * (7 - 4) // -> 35

10 / 3 // -> 3.33333...
2.5 * 7.3 // -> 18.25

It also supports some more complicated operations, such as % and **, which are modulo and exponentiation.

Note: Modulo is defined as remainder after division, which can be useful, for example, to determine if a number is even.

22 % 7 // -> 1
5 % 2 // -> 0
12 ** 3 + 1 ** 3 // -> 1729

Using the Math library, other important functions can also be used.

Math.PI // -> 3.14159...
Math.sqrt(24) // -> 4.89897...

Math.random() // -> 0.86235...
Math.random() // -> 0.98774...
Math.random() // -> 0.63295...

Variables

In JavaScript, variables can be declared with let. Variable names can be any combination of letters and numbers, but cannot start with a number.

Note: A variable cannot be used before it is declared.

let wheels = 4
wheels * 5 // -> 20

seven * 5 // -> ReferenceError: seven is not defined
let seven = 7

You can also use basic mathematical operators to update variables by appending an = to the operator. As incrementing and decrementing are common operations, you can use the shorter ++ and -- operators.

let n = 1
n += 2 // -> 3
n--
n // -> 2
n *= 6 // -> 12
n /= 3 // -> 4
n %= 3 // -> 1

Variables can also be declared with const to make them constant. This is useful for variables that must be used in multiple files. It is also convention for the name to be capitalised.

const DNA_STRANDS = 2
DNA_STRANDS = 3 // -> TypeError: Assignment to constant variable

To represent values of yes/no or on/off, we can use true and false.

We can also use Boolean operations: ! for negation, && for AND and || for OR.

let computerIsOn = true
let computerIsOff = !computerIsOn
computerIsOff // -> false

Objects

Objects can be declared by using curly brackets. To check if an object has a property, you can use object.hasOwnProperty(property).

let ferrari = { wheels: 4, zeroToSixtyIn: 3.5 }
ferrari.isFast = true
ferrari.wheels // -> 4
ferrari.hasOwnProperty("isCamel") // -> false
ferrari // -> {wheels: 4, zeroToSixtyIn: 3.5, isFast: true}
