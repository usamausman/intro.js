HTML

DOM

HTML is structured with a Document Object Model, or DOM for short. This dictates the parent and child elements, as well as their styling and attributes. To select elements on an HTML page, you need to use their CSS selectors. For example, to select all links with an external class, you can use a.external.

let title = document.querySelector("section#dom h2")
title.innerText = "DOM Updated" // -> updates this section's title

Using querySelectorAll, we can update multiple elements at once.

Note: Use Array.from to create arrays from 'almost arrays', such as those returned from document.querySelectorAll.

let titles = Array.from(document.querySelectorAll("section h2, section h3"))
titles.forEach((el) => el.style.color = "red")
// -> changes color of all titles on this page to red

Classes

To change the classes on an element, we can use its classList property, with which we can add, remove and toggle classes.

document.body.classList.toggle("cs30")

Event Handlers

JavaScript is also used to respond to events on a webpage, such as hover or click elements.

Note: Events can be scheduled for the future or on set intervals using the setTimeout and setInterval functions.

let title = document.querySelector("section#eventhandlers h2")
let greeter = (e) => {
  console.log("Hello from intro.JS!")
}

console.log("[adding]")
title.addEventListener("click", greeter) // -> now click on the 'Event Handlers' text

setTimeout(() => {
  console.log("[removing]")
  title.removeEventListener("click", greeter)
}, 5000) // -> removes click handler after 5 seconds
