Loops

while loops

To run code multiple times, we can use loops. A while loop will run until a specified condition is false. This can be useful with counters or when reading files.

let counter = 1
let sum = 0

while (counter <= 10) {
  sum += counter
  counter++
}

console.log(sum) // -> 55
 Note: if your code gets stuck in an infinite loop, make sure you are updating the values of the condition.

for loops

A for is a shorter version of a while loop. This helps to keep the condition and update statements in one place. The format consists of three statements separated by ;s.

for (initialisation; condition; update) 

let sum = 0

for (let counter = 1; counter <= 10; counter++) {
  sum += counter
}

console.log(sum) // -> 55

for of loops

A for of is a cleaner way to go through the elements of an array.

const pie = ["May", "I", "have", "a", "large", "container", "of", "coffee", "cream", "and", "sugar"]

for (let word of pie) {
  console.log(word.length)
}
// -> 3
// -> 1
// -> 4
// -> 1
// -> 5
// -> 9
// -> 2
// -> 6
// -> 5
// -> 3
// -> 5

Using with Arrays

If you need to access an array's elements by index, you can use a for loop to do this. This can be useful for some situations, such as for sorting algorithms.

Note: you can use graves `` to interpolate strings and use ${} to substitute values into the string.

let n = 5
let numbers = [5, 7, 3, 4, 2, 1, 6]

let example = `n is ${n} and the array length is ${numbers.length}`
console.log(example) // -> "n is 5 and the array length is 7"

for (let i = 0; i < numbers.length; i++) {
  console.log(`[${i}] is ${numbers[i]}`)
}

// -> [0] is 5
// -> [1] is 7
// -> [2] is 3
// -> [3] is 4
// -> [4] is 2
// -> [5] is 1
// -> [6] is 6
