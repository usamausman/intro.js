Arrays

Syntax

Arrays can be declared in JavaScript using []. Arrays can hold values of any type, and elements can be accessed by using their index.

Note: Computers starts counting at 0, so the first element is at index 0.

let squares = [1, 4, 9, 16, 24]
console.log(squares[0]) // -> 1
console.log(squares[2]) // -> 9
squares[4] = 25
console.log(squares[5]) // -> undefined
console.log(squares) // -> [1, 4, 9, 16, 25]

Values can be added to the beginning and end of arrays using push and unshift.

let numbers = [2]
numbers.unshift(1)
numbers.push(3)
console.log(numbers) // -> [1, 2, 3]

Methods

To run a function on every element in an array, we can use the forEach function function, which takes a function as an argument and runs it on every element in an array.

Note: console.log is a useful way to print to the console.

let numbers = [1, 2, 3, 4, 5, 6]

numbers.forEach((n) => console.log(n))
// -> 1
// -> 2
// -> 3
// -> 4
// -> 5
// -> 6

To run a function on every element in an array and create a new array of the output, we can use the map function function, which takes a function as an argument, maps it over the array and returns a new array.

const square = (n) => n * n
let numbers = [1, 2, 3, 4, 5, 6]

console.log(numbers.map(square)) // -> [1, 4, 9, 16, 25, 36]
console.log(numbers) // -> [1, 2, 3, 4, 5, 6]

To run a function on only some elements of an array, we can use the filter function function, which takes a function as an argument, uses it to filter the array and returns a new array.

let numbers = [1, 2, 3, 4, 5, 6]
const isEven = (n) => n % 2 == 0
console.log(numbers.filter(isEven)) // -> [2, 4, 6]
console.log(numbers) // -> [1, 2, 3, 4, 5, 6]

To reduce an array to a single value, we can use the reduce function, which takes a function as an argument, uses it to reduce the array and returns a single value.

let numbers = [1, 2, 3, 4, 5, 6]
const sum = (total, n) => total + n
console.log(numbers.reduce(sum, 0)) // -> 21

Combining all these ideas, we can create powerful code.

let numbers = [1, 2, 3, 4, 5, 6]
const isEven = (n) => n % 2 == 0
const square = (n) => n * n
const sum = (total, n) => total + n
// take the even numbers, square them and take their sum
console.log(numbers.filter(isEven).map(square).reduce(sum, 0)) // -> 56

Strings

In JavaScript, characters and sentences are represented by strings. Characters in the string can be referred to by their index, similar to arrays. You can also get a string's length with the string's length property.

const hello = "Hello, World!"
console.log(hello) // -> "Hello, World!"
console.log(hello[0]) // -> "H"

console.log(hello.length) // -> 13
console.log(hello[hello.length - 1]) // -> "!"

Note: the indexes of the characters in hello start from 0, so the last character is at hello.length - 1.

You can also extract substrings from a string using the substring function, which takes a start index and an optional end index. The string returned will be from the start index up to, but not including the end index. If an end index is not provided, it will go to the end of the provided string.

Strings can also be added together, or concatenated, using a +.

const hello = "Hello, World!"
      console.log(hello.substring(7)) // -> "World!"
console.log(hello.substring(3, 10)) // -> "lo, Wor"
console.log(hello.substring(0, 5)) // -> "Hello"
console.log(hello.substring(0, 5) + hello.substring(12)) // -> "Hello!"
