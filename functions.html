Functions

Definition

To split up multiple calculations into separate functions, they can be defined in JavaScript with a name and a list of arguments:

const square = (n) => { // name and an argument 'n'
  return n * n
}

const addOne = (n) => n + 1 // shorter way to declare a function
//                   ^ return is implied here

square(3) // -> 9
square(5) // -> 25
addOne(6) // -> 7

Functions can also be chained to run them in a sequence:

square(square(3)) // -> 81

addOne(square(3)) // -> 10
square(addOne(3)) // -> 16

Multiple Arguments

Functions can take multiple arguments, but can only return a single output.

const leftRoot = (a, b, c) => (-b - Math.sqrt(b ** 2 - 4 * a * c)) / (2 * a)
const rightRoot = (a, b, c) => (-b + Math.sqrt(b ** 2 - 4 * a * c)) / (2 * a)

leftRoot(1, -2, -3) // -> -1
rightRoot(1, -2, -3) // -> 3

To return multiple outputs, they can be put into an object or an array, which is then returned.

const solveQuadratic = (a, b, c) => {
  const discriminant = Math.sqrt(b ** 2 - 4 * a * c)
  let a = (-b - discriminant) / (2 * a)
  let  b = (-b + discriminant) / (2 * a)
  return { a, b }
}

solveQuadratic(1, -2, -3) // -> {a: -1, b: 3}
